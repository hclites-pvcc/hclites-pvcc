Networking & Internet Terminology
TERM DEFINITION
Network Classifications
  • LAN: local area network
  • MAN: metropolitan area network
  • WAN: wide area network
Network Models
  • bus network: a network in which machines are all connected to a common communication line called a bus
  • star network: a network in which one machine serves as a central focal point to which all others are connected
Protocols
  • CSMA/CD Ethernet: Carrier Sense, Multiple Access with Collision Detection - a protocol that dictates that each message be broadcast to all machines on the bus
  • CSMA/CA Wifi: Carrier Sense, Multiple Access with Collision Avoidance - a protocol that is designed to avoid collisions rather than trying to detect them
Combining networks
  • repeater: a device that passes signals back and forth between the original two buses
  • bridge:
  • switch: a bridge with multiple connections
  • router: special purpose computers used to forward messages by providing links between networks while keeping each network's properties distinct from one another
  • gateway: the point at which one network is linked to an internet, serving as the passageway between the network and the outside world
Client/server model
  • P2P: peer-to-peer: a model for communication that involves processes that give and receive service to/from each other, usually executing on a temporary basis
  • distributed system: a system consisting of software units that execute as processes on different computers
  • cloud computing: a distributed system where huge pools of shared computers on a network can be used by clients on an as needed basis
Internet Addressing
  • Tier 1: networks that consist of very high speed and high capacity international WANs
  • Tier 2: networks that are somewhat more regional than Tier 1 and with less capabilities
  • Tier 3: also known as an access ISP or an intranet - an access point provided by a business that supplies internet access
IP address
  • ICANN:
  • DNS: domain name system - an internet wide directory system thatr provides address translation services to clients for domain names to IP addresses
  • TLD: Top-level domain: the suffix of a domain name, e.g. *.edu, *.com, *.net, *.org, etc.
  • Country-code TLDs: two letter TLDs specific for each country, e.g. *.au for Australia
Internet protocols
  • FTP: File Transfer Protocol - an application used to access another computer from a distance
  • HTTP: Hyper Text Transfer Protocol - became more utilized as web servers and browsers have become more sophisticated
  • SMTP: Simple Mail Transfer Protocol - a way that two computers may interact when sending email messages to each other
World Wide Web
  • URL: Uniform Resource Locator - a unique Web address given to documents
  • client-side activities: activities performed by a client, such as a browser
  • server-side activities: activities performed by a server, such as a web server
Internet four-layer software
  • application layer: consists of software units such as clients and servers that execute tasks via internet communication
  • transport layer: accepts messages from the application layer and makes sure the messages can be transmitted over the internet
  • network layer: receives segments of a message known as packets from the transport layer, and sends them along as separate segments
  • link layer: transfers the packet(s) of the message based on the network in which the computer resides
Internet is
  • packet-switching network: a system by which data is sent by breaking it down into smaller pieces called packets
  • fault tolerant: resistant to failures
  • redundant: a property of routing in which there is more than one way to route data
Cybersecurity
  • malware: short for malicious software
  • virus: software the infects existing programs in a machine, so that the virus software forcibly executes
  • worm: an autonomous program that transfers itself through a network by making replicas of itself
  • Trojan horse: a program that enters a machine disguised as a desirable software that the victim chooses to install, but is actually harmful
  • spyware: malware that collects information about a computer's activities
  • phishing: obtaining information from someone by asking for it, under the guise that it is being requested for legitimate reasons
  • DDoS attack: the process of overloading a computer with messages
Protection
  • firewall: a program that filters incoming traffic to protect from malicious attacks
  • spam filter: firewalls design to block unwanted emails
  • proxy server: a software unite that acts as an intermediary between a client and a server to protect the user from attacks originating from the server
